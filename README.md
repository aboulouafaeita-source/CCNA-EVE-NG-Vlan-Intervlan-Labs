# CCNA EVE-NG Labs

Hands-on networking labs built in EVE-NG for CCNA practice, troubleshooting, and documentation.

## About This Repository
This repository contains my networking labs built in EVE-NG as part of my CCNA learning journey.

Each lab includes:
- Topology screenshots
- Configuration files
- Verification results
- Notes and troubleshooting steps

## Current Labs

- VLAN configuration
- Access and trunk ports
- Inter-VLAN routing
- IP addressing
- Connectivity testing
- Basic troubleshooting

## Tools Used

- Cisco CLI
